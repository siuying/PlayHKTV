Play HKTV
=========

Unofficial app to watch HKTV on your Mac.

How to Install
--------------

Download the zip file, unzip it and drag "Play HKTV" to your Applications folder. 

Requirement
-----------

- OS X 10.10 (Yosemite), should work on 10.9 (Mavericks).
